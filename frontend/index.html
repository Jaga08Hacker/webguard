    About WebGuard

    WebGuard is an automated, passive data-exposure detection tool built for security researchers, ethical hackers, and system administrators. It uses OSINT techniques to detect publicly accessible sensitive information — without attacking or harming the target system.

    🌐 Subdomain Enumeration

    Uses crt.sh, HackerTarget, and DNS brute-force to find all subdomains associated with a domain.

    🔍 Google Dorking

    Searches for exposed files, credentials, admin panels, and sensitive data indexed by Google.

    📡 Shodan Integration

    Identifies open ports, services, and misconfigurations using the Shodan internet-wide scanner.

    ☁️ S3 Bucket Detection

    Checks for publicly accessible Amazon S3 buckets using common naming patterns for the domain.

    🕵️ Hidden Path Scanner

    Discovers exposed admin panels, config files, backup files, and debug endpoints via HTTP probing.

    ⚠️ CVE Mapping

    Maps detected services and software to known CVEs from NIST NVD and Shodan vulnerability data.

    ⚠ Legal Disclaimer: WebGuard is intended for educational use, authorized security auditing, and ethical research only. Only scan domains you own or have explicit permission to test. Unauthorized scanning may violate laws.
